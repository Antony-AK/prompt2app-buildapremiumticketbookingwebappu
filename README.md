# Ticket Booking App

A premium ticket booking web app UI built with React and Tailwind CSS.

## Features

- Modern event listing with cards
- Event detail view with seat selection
- Booking confirmation flow
- Responsive design
- State management without routing

## Development

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
